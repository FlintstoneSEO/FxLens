# FxLens

FxLens is an AI workspace for Power Apps teams.

## Phase 1 (current)

This repository now includes the initial Next.js + TypeScript + Tailwind foundation and a product-shell homepage covering:

- Scope Studio
- Build Studio
- Analyze Studio

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
